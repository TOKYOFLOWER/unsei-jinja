# 運勢神社

日本の伝統的な暦と九星気学に基づく、和風デザインの運勢サービスです。

🌸 公開URL: https://jinja.tokyoflower.jp/

## できること

- **今日の運勢** — 全体運・恋愛・仕事・金運・健康を、AI を使って毎日お届けします
- **吉方位コンパス** — 24方位を可視化し、本日の吉凶を一目で確認できます
- **行動表** — 朝・正午・午後・夕の時間帯別に、推奨される行動をご案内します
- **月の運勢** — 1ヶ月先までの吉日早見カレンダー
- **本命星設定** — 生年月日から立春境界を考慮した自動算出

## ご利用方法

ブラウザで https://jinja.tokyoflower.jp/ にアクセスし、生年月日を入力するだけでご利用いただけます。生年月日と本命星はお使いの端末内にのみ保存され、サーバーには送信されません。
